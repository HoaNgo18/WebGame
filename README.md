# Sword IO – Web Game Online

Dự án game **.io** chém kiếm thời gian thực, lấy cảm hứng từ **evowar.io** (mục tiêu học tập & mở rộng).

## Tính năng chính
- Di chuyển mượt mà, phản hồi tức thì
- Chiến đấu PvP thời gian thực
- Thu thập điểm số & tiền tệ trong game
- Hệ thống nâng cấp kiếm, tốc độ, sức mạnh
- Bảng xếp hạng toàn cầu & theo phiên
